# Login Screen React

Projeto desenvolvido para praticar React, componentização e manipulação de formulários.

## Tecnologias
- React
- JavaScript
- CSS

## Funcionalidades
- Campo de e-mail
- Campo de senha
- Validação básica
- Interface responsiva

## Como executar

npm install
npm run dev
